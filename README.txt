# Coke vs Pepsi — Price & Returns Comparison

Notebook comparing Coca‑Cola (KO) and Pepsi (PEP) from 2020-10-01 to 2025-10-01 using yfinance.
How to run:
1. Create a Python venv, install dependencies from requirements.txt.
2. Open the notebook and run cells.
Notes: notebook downloads live data from yfinance.
